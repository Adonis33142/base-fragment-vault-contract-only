# BaseSplitVault

BaseSplitVault is a contract-only project hosted at:

https://github.com/Adonis33142/base-fragment-vault-contract-only.git

## Overview

BaseSplitVault provides a focused repository for contract development.

The project keeps the scope narrow by concentrating on contract-related code rather than a full application stack.

Use this repository as the primary reference for the BaseSplitVault contract implementation, configuration, and related documentation.
