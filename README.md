# BaseSplitVault

BaseSplitVault is a contract-only project.

Repository: https://github.com/Adonis33142/base-fragment-vault-contract-only.git

## Overview

BaseSplitVault provides a focused repository for contract development.

The project is intentionally limited to contract-related source code, configuration, and supporting files.

It is not intended to be a full application stack with frontend or backend application code.

Use this repository as the primary reference for the BaseSplitVault contract implementation and related project files.
